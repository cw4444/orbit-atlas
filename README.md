# Orbit Atlas

Orbit Atlas is a small atmospheric workspace for tracking thoughts, moods, and momentum as a living orbital map.

It now remembers your notes and mood in the browser using `localStorage`.

## Built by Codex

This project was built entirely by [Codex](https://chatgpt.com/codex/cloud) from OpenAI.
## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
